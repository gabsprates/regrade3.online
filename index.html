Entenda como funciona a
Regra de 3

Primeiro de tudo: a regra de 3 é uma continha usada para o cálculo de proporções!

A ideia é encontrar um valor qualquer (podemos chamar de x), baseado em outros 3 valores que você já tenha em mãos.

Temos que pensar em equivalências. Precisamos identificar quais valores são equivalentes.

Por exemplo, se você sai com 3 pessoas pra comer pizza e a conta dá R$ 64,00, quanto cada um terá que pagar?

Aí é que precisamos saber quais as grandezas (ou valores) são proporcionais: 

4 pessoas (você + 3) -> R$ 64,00
1 pessoa             -> ?

E então, podemos dizer que a equivalência é: 

 4  =  64,00 
 1       x

Resolvendo a equação, temos que cada pessoa terá que pagar R$ 16,00 💸

Aproveitando o exercício, você pode dizer quantas fatias cada pessoa irá comer, considerando que peçam uma pizza de 8 fatias?

Considere um novo exemplo: pensando que uma pessoa gasta 3 horas para fazer uma viagem de carro em uma velocidade de 90km/h, quanto tempo essa pessoa gastaria se a velocidade fosse 110km/h?

Lembrando dos passos que seguimos, primeiro vamos achar os valores proporcionais: 

90km/h  -> 3 horas
110km/h -> x horas

Observe o que acontece se fizermos a equação de equivalência dessa forma: 

  90  =  3 
 110     x 

     👇

x =  3 * 110 
       90

     👇

x = 3,666... (~ 3h 36min)

ESPERE UM POUCO!!! Este resultado está errado! Se aumentarmos a velocidade deveria diminuir o tempo... O que pode estar errado? 😰

Nesse caso, precisamos ter atenção para encontrar os valores proporcionais corretamente. Sabemos que ao aumentar a velocidade, o tempo final deve DIMINUIR, já que está indo mais rápido.

Sabendo disso, estamos em uma situação de grandezas (ou valores) inversamente proporcionais. Nesse caso, precisamos inverter os valores de um dos lados da equivalência. O que era: 

90km/h  -> 3 horas
110km/h -> x horas

Passa a ser (invertendo um dos lados): 

110km/h -> 3 horas
90km/h  -> x horas

E assim podemos seguir com a montagem da equivalência e resolução da equação: 

 110  =  3 
  90     x 

   👇

110 * x = 3 * 90

   👇

x =  3 * 90 
      110

   👇

x = 2.45... (~ 2h 27min)

Agora sim!!!

Você acabou de ver como fazer regra de 3 para valores diretamente e inversamente proporcionais!

Para fechar, considere mais uma situação:

Exemplo: Se cada pessoa que compartilha o regrade3.online em uma das redes sociais faz com que 3 novas pessoas entrarem no site, quantas pessoas novas vão entrar no site se 123.456 pessoas compartilharem?

Passo 1: Identificação das grandezas
1 pessoa equivale à 3 novas pessoas, assim como 123.456 pessoas equivalem à x novas pessoas.

Passo 2: Montagem da equivalência 

   1    -> 3
123.456 -> x

      👇

   1    =  3 
123.456    x

Passo 3: Resolução da equação

   1    =  3 
123.456    x 
              
      👇

   1    =  3 * 123.456 
                x 
              
      👇

 1 * x  =  3 * 123.456 
              
      👇

 1 * x  =  370.368

Resposta: 370.368 novas pessoas 👏👏👏
